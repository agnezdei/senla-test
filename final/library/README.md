## Требования
- Java 17, Maven 3.8+, Docker & Docker Compose

## Установка и запуск

1. Распаковать архив.
2. Перейти в корень проекта.
3. Выполнить: `\run.sh` (поднимает БД и приложение).
   Приложение будет доступно по адресу: http://localhost:8080

## Тестирование
`mvn test` – запуск модульных тестов

## Миграции БД
Flyway – скрипты в `src/main/resources/db/migration`

## Роли
- USER: может брать/возвращать книги, просматривать свой профиль.
- ADMIN: полный CRUD над книгами, экземплярами, каталогами, управление пользователями.
